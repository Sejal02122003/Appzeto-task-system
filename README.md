# Appzeto-task-system
Built a modern Kanban sprint board web app using React + Vite + TypeScript.
Imported and cleaned task data from a JSON file to display real sprint tasks.
Added a professional dashboard with:
Overview
Sprint Board
Reports
Teams
Calendar
Included drag-and-drop task movement across Kanban columns.
Added useful sprint insights like:
progress chart
overdue tasks
total effort
completion stats
Added a “Suggest Next Task” feature to recommend the most urgent task.
Included a clean, futuristic UI with Tailwind CSS  and motion effects.
Added reset and history support for better task board control.
Pushed the project to GitHub for easy sharing and deployment.
